1. Enter RNA Sequence

How to Format Your Sequence

  • Enter the 5' to 3' sequence using standard bases (A, U, C, G).
  • The tool calculates for a 5'-monophosphate and 3'-hydroxyl terminus by default.
  • For modified bases, use brackets `[]`. For example, for a Methyl-6-Adenosine, write `A[m6A]`.
  • Make sure the symbol you use in the sequence (e.g., `m6A`) matches a symbol defined in the Modifications section.
  • Any characters not in `AUCG[]` or defined as a custom modification will be flagged as an error.
